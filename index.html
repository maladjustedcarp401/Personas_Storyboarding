Interface Sketch!


Figure 1:
  • The interface’s purpose is to allow users of the 3D printer a visual way to print their files, calibrate the printer, and edit the printer's configuration/settings all on the printer itself without needing another computer or device.
    • The knob allows you to scroll between options, but is also the “Ok” button as it can be pushed down
    • The X button is a reset button, resetting the printer to the main menu

Part 2

Some user observations include:

  • Was not clear that the text in the “Print from SD” option are file names. Many file names also do not fit in the screen
  • Most new users of the printer (and experienced 3D printer users) got stuck on the main menu for a while because they did not realize the knob was also a button
  • Users became concerned on the main menu because when turning the knob it would change some of the numbers of the main menu, but not really indicate what exactly was being changed
    • This usually happened when users were trying each component hoping one of them would leave the main menu (before they realized the knob could be pushed down)
    • It seems to be changing target temperature
  • Once in the options menu, there was confusion between scrolling to the “back” option and pressing the knob button vs hitting the reset button
    • Users would first hit the reset button, bringing them back to the main menu after a loading screen
    • This annoyed a lot of the users as they had to start over from where they were before
  • The X button has become the “panic” button to press when anything with the U.I or print goes wrong

Numbered User Questions With Summary of Answers

  1. How many times have you printed using a 3D printer before, and do you think it has made your experience easier?
  2. How did you originally learn to use the 3D printer, and were you able to retain a lot of that information when using the printer again? Or do you find yourself getting stuck on similar parts again?
  3. Did you find the knob to be a good fit to the LED interface?
  4. Could you describe what the X button does?
    • If you aren’t sure, could you describe what your first intuition would be
  5. Do you think the knob having 3 extrusions helps with your understanding or adds confusion?
  6. When your print has issues how does the interface help you diagnose the problem?
  7. What part about the Main Menu is the most confusing or unfamiliar to you

  • Generally having been taught from a BDW mentor or having used another 3D printer before helped with using the printer, but surprisingly a lot of the following issues still happened to these users (1 & 2)
  • Generally for users both familiar with 3D Printers and unfamiliar the X button was a pain point. Users were confused if it was a back button or ok button and wasted a lot of time clicking it. It is actually a reset button which barely ever has to be pressed (4)
  • When trying to leave the main menu users ended up using the reset button above or turning the knob because it was not obvious that the knob was a button. It took users about a minute to realize that it was a button
  • When a print is having issues, the user's first instinct is to hit the X button. This makes it had to diagnose a problem because it goes to the main menu, and there are other ways to safely stop a print that are not obvious (6)
  • Users found the main menu to be cluttered with text and numbers, and it was not clear how the knob affected them. Experienced users were more comfortable with ignoring this screen, whereas new users became very worried. (3 & 7)
